Go — это язык программирования с **статической типизацией**, что означает, что каждая переменная имеет фиксированный тип, который нельзя изменить после назначения.

| uint         | Максимальное значение (32/64)Bit  | Зависит от разрядности системы | 0                |

**Объявление переменной:**
---



```go
var назв тип
```

Пример в программе

```go

package main

import (
	"fmt"
)
var a int = 8 // значение a будет равно 8
fmt.Println(a) // выведет 8

```


**Константы**
---

Константы, как и переменные, хранят данные, но в отличие от переменных, значения констант нельзя изменить после их определения.


```go
const a 8
_______________________________________________________________________________________________________________________________________________________________________________

const (
    Sunday = 1
	Monday = 2
	Tuesday = 3
	Wednesday = 4
	Thursday = 5
	Friday = 6
	Saturday = 7
)

func main() {
	fmt.Println(Sunday)   // вывод 0
	fmt.Println(Saturday) // вывод 6
}

```

Так же можно использовать **iota**

**iota** идентификатор Go используется в объявлениях констант для упрощения определений увеличивающихся чисел.  
Сделаем дни недели с использованием **iota -** теперь это выглядит проще (особенно если много данных)  

```go

const (
	Sunday = iota
	Monday
	Tuesday
	Wednesday
	Thursday
	Friday
	Saturday
	_ // пропускаем 7
	extra
)

func main() {
	fmt.Println(Sunday)   // вывод 0
	fmt.Println(Saturday) // вывод 6
	fmt.Println(extra) // вывод 8
}

```
